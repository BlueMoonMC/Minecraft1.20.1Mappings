**Minecraft Forge Mappings for The Minecraft Version 1.20.1**

#### Warning: these mapping ONLY Work for the Minecraft Version 1.20.1 and ONLY Forge not neoforge or fabric or anything else

### How to Use it:

- Download the Forge mod you want to implement in to your Client.
- Decompile it with a Decompiler i recomend this one http://www.javadecompilers.com.
- if you have a Client setet up go to the next step else i would recomend setting up MCP Reborn after this tutorial https://github.com/Hexeption/MCP-Reborn.
- open it with a Java IDE i recomend Intellij another known one is Eclipse.
- open the mapping files with a text programm like exel or Numbers on Macos.
- now copy the Number wich is not known in the mod
- search it in the mapping files by pressing strg F or cmd f on Macos.
- copy the word standing on the other side and paste it where your old Number was standing.

- i will descripe how to implement it in a little text because its to long to write it here
- open the mixm files and copy the code unter the mixims you can comment the mixims out by doing //
- now open the class wich is written there like playerenderer.class remove the class and open the playerenderer class in your Client.
- paste the code into the playerenderer and do the same thing with every other mixim
